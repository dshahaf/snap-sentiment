//////////////////////
// Scoring function
//////////////////////
p: number of positive neighbors
n: number of negative neighbors
k: scale factor

total number of neighbors factor ([0, ]):
log(1 + p + n)

difference factor ([0, 1]):
1 - (|p - n| / (1 + p + n))

difference factor scaled by k ([1, k]):
((1 - (|p - n| / (1 + p + n))) * (k - 1) + 1)

final score:
log(1 + p + n) * ((1 - (|p - n| / (1 + p + n))) * (k - 1) + 1)
